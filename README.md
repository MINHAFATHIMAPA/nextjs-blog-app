# 🚀 Next.js Blog Application

A modern full-stack blog application built using Next.js with dynamic routing, interactive UI, and structured content rendering.

---

## ✨ Features

- 📖 Dynamic blog pages (`/blog/[id]`)
- 🖼️ Image carousel with navigation
- 🎥 Embedded YouTube videos inside articles
- 📌 Sidebar with related articles
- ❓ FAQ system with expand/collapse interaction
- 🧠 Smart content rendering (headings, lists, tables, flashcards)
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

- Next.js
- React
- Tailwind CSS
- JavaScript
